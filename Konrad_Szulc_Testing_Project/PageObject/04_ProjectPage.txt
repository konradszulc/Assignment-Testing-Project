# Positive Test examples are below write test process steps for each below test case

	Create blank project
		Navigate to Create New Project Page ProjectPageURL (https://gitlab.com/projects/new)
		Click Create Blank Project
		Redirect to Create Blank Project page
		Input Project Name BlankProjectName (First Project)
		Input User into Project URL UserName (KSlab2025)
		Input Project Slug ProjectSlug (first-project)
		Input Project Deployment Target Empty
		Click Visibility Level Private
		Untoggle Initialize Repository with a README
		Click Create Project
		Verify Message is showing Project 'First Project' was successfully created
		Expected output page contains first-project page

	Create template project
		Navigate to Create New Project Page ProjectPageURL (https://gitlab.com/projects/new)
		Click Create from template
		Redirect to Create from template page
		Click Use Template for NodeJS Express section
		Input Project Name TemplateProjectName (Second Project)
		Input User into Project URL UserName (KSlab2025)
		Input Project Slug TemplateProjectSlug (second-project)
		Input Project description ProjectDescription (Testing123)
		Click Visibility Level Private
		Click Create Project
		Verify Message The project was successfully imported
		Expected output page contains second-project page with NodeJS Express Template

	Update name of template project
		Navigate to Template Project page PersonalProject2PageURL (https://gitlab.com/KSlab2025/second-project)
		Click Settings
		Click General
		Redirect to Edit Page
		Input Project Name ReTemplateProjectName (Second Project-02)
		Click Save Changes
		Verify Message Project 'Second Project-02' was Successfully Updated
		Expected output Template Project Name Updated 
		

	Delete template project
		Navigate to Template Project page PersonalProject2PageURL (https://gitlab.com/KSlab2025/second-project)
		Click Settings
		Click General
		Redirect to Edit Page
		Click Advanced
		Click Delete 
		Input Delete Confirmation DeleteConfirm (KSlab2025/second-project)
		Click Yes, delete project
		Verify Message This project is pending deletion, and will be deleted on 2026-01-07. Repository and other project resources are read-only.
		Expected output Template Project to be pending deletion
	
	(bonus) Delete pending template project
		Navigate to Template Project page PendingProject2DeletionURL (https://gitlab.com/KSlab2025/second-project-deletion_scheduled-76957501)
		Click Settings
		Click General
		Redirect to Edit Page
		Click Advanced
		Click Delete Immediately
		Input Delete Confirmation PermDeleteConfirm (KSlab2025/second-project-deletion_scheduled-76957501)
		Click Yes, delete project
		Verify Message Second Project-02 has been permanently deleted
		Expected output Template Project no longer available in profile
